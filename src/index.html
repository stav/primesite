Technology

Progressive Web Apps

Say goodbye to apps. Thanks to the all-new background process architecture, the browser is now both safer and faster than native apps on your phone. Which means we don't need Google Play Store or Apple App Store any more, and best yet, these new processes work on the desktop too! We're just using the web browser after all.

Responsive Design

Mobile-first is a smart style commitment not only because it is the fastest growing user agent market and of course all professional sites need to be responsive to different device sizes. It is smart because the limited space on a small view-port makes the screen real estate more valuable and therefore the user interface better by design.

Single Page Applications

Unleash the power of offline. SPA's send everything to your browser in a single payload making user interaction lightning fast. Combined with Progressive Web technology you can even allow users to create database transactions while offline and have them committed later when the user attaches to the network.

Data Mining

Spiders bring back the goods. Whether you want to gather product information from all competitor websites or crawl the "entire" world wide web to have your own index similar to Google's or maybe you just want search engine results every day for mention of your name, we've got the tooling and infrastructure to get your bots out, crawling and bringing back the data.
